# ООО «Обувь» — экзаменационный проект по модулям 1–3

Готовый проект по КОД 09.02.07-2-2026 для сдачи ссылкой на GitHub.

## Что реализовано

- база данных SQLite в 3НФ с внешними ключами;
- импорт исходных данных из файлов `resources/imports`;
- авторизация с ролями: гость, авторизированный клиент, менеджер, администратор;
- окно входа с переходом в режим гостя;
- список товаров с выводом фото, описания, поставщика, производителя, цены, единицы измерения, остатка и скидки;
- подсветка товаров по условиям задания:
  - скидка больше 15% — фон `#2E8B57`;
  - если есть скидка — старая цена перечёркнута красным, рядом выводится итоговая цена;
  - если товара нет на складе — строка голубого цвета;
- поиск по всем доступным текстовым полям;
- фильтрация по поставщику с первым значением `Все поставщики`;
- сортировка по количеству на складе по возрастанию и убыванию;
- поиск, фильтрация и сортировка работают в реальном времени;
- добавление, редактирование и удаление товара для администратора;
- ID товара:
  - при добавлении не отображается и вычисляется автоматически;
  - при редактировании доступен только для чтения;
- загрузка и замена изображения товара с ограничением 300x200;
- запрет удаления товара, если он присутствует в заказах;
- отдельная страница заказов:
  - менеджер может просматривать заказы;
  - администратор может просматривать, добавлять, редактировать и удалять заказы;
- обработка ошибок через информационные, предупреждающие и ошибочные сообщения;
- PDF с ER-диаграммой;
- PDF с блок-схемой алгоритма;
- DOCX со скриншотами работы приложения.

## Быстрый запуск

```bash
pip install -r requirements.txt
python init_db.py
python app.py
```

## Логины для проверки

Данные загружаются из `resources/imports/user_import.xlsx`. Можно использовать любую запись из этой таблицы.

## Дамп базы данных

Находясь в папке проекта:

```bash
python -m sqlite3 db.sqlite3 ".dump" > dump.sql
```

Готовый дамп уже приложен в файле `dump.sql`.
