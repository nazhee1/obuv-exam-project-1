# ООО «Обувь» — экзаменационный проект по модулям 1-3

Исправленный проект по заданию `КОД 09.02.07-2-2026` для предметной области `ООО «Обувь»`.

Что реализовано:

- авторизация по пользователям из исходного `users.xlsx`;
- вход в роли гостя;
- каталог товаров для гостя и авторизированного клиента без поиска, фильтрации и сортировки;
- каталог товаров для менеджера и администратора с поиском, фильтрацией и сортировкой;
- просмотр заказов менеджером;
- CRUD товаров и заказов для администратора;
- SQLite база данных с внешними ключами;
- автоматическое заполнение БД из исходных файлов `products.xlsx`, `users.xlsx`, `orders.xlsx`, `pickup_points.xlsx`;
- стиль интерфейса по модулю 2:
  - шрифт `Times New Roman`;
  - белый основной фон `#FFFFFF`;
  - дополнительный фон `#7FFF00`;
  - акцент `#00FA9A`;
  - товары со скидкой больше `15%` подсвечиваются цветом `#2E8B57`;
- логотип и иконка из приложенных ресурсов;
- скрипт для формирования `dump.sql`.

## Структура проекта

- `main.py` — запуск приложения;
- `src/obuv_app/database.py` — схема БД, импорт данных, CRUD, формирование `dump.sql`;
- `src/obuv_app/ui.py` — интерфейс на `tkinter`;
- `resources/imports/` — исходные файлы задания и нормализованные копии `xlsx`;
- `assets/` — иконка, логотип, фотографии товаров;
- `scripts/reset_database.py` — полное пересоздание базы;
- `scripts/create_dump.py` — формирование `dump.sql`.

## Запуск

Нужен Python 3.11+.

```bash
python main.py
```

Если команда `python` не работает в терминале, используйте путь к вашему интерпретатору Python.

## Команды для сдачи

Пересоздать базу:

```bash
python scripts/reset_database.py
```

Сформировать `dump.sql`:

```bash
python scripts/create_dump.py
```
